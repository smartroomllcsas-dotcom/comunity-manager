# Client Onboarding Checklist

**Client:** ___  
**Start Date:** ___  
**Account Manager:** ___

---

## Phase 1: Discovery

- [ ] Client name confirmed
- [ ] Industry confirmed
- [ ] Website URL collected
- [ ] Existing brand materials received
- [ ] Content samples collected (3+)
- [ ] Target language defined

## Phase 2: Positioning

- [ ] Question 1 answered: What do you do?
- [ ] Question 2 answered: Who do you serve?
- [ ] Question 3 answered: What problem do you solve?
- [ ] Question 4 answered: What makes you different?
- [ ] Question 5 answered: What is your proof?
- [ ] One-liner drafted
- [ ] One-liner approved by client

## Phase 3: Voice

- [ ] Voice extraction completed (3+ samples analyzed)
- [ ] Tone defined
- [ ] Personality traits identified (3-5)
- [ ] Do list documented (5+ items)
- [ ] Don't list documented (5+ items)
- [ ] Example phrases documented

## Phase 4: Brand Identity

- [ ] Primary color defined
- [ ] Secondary color(s) defined
- [ ] Fonts selected
- [ ] Visual style defined
- [ ] Photo style defined

## Phase 5: Strategy

- [ ] Content pillars defined (3-5 with % allocation)
- [ ] Active platforms selected with posting frequency
- [ ] Hashtag strategy created
- [ ] Goals defined
- [ ] KPIs established

## Phase 6: Competition

- [ ] 3+ competitors identified
- [ ] Positioning mapped against competitors
- [ ] Competitive advantages documented

## Phase 7: Profile

- [ ] `.clients/{slug}/context.md` created
- [ ] All 10 sections filled
- [ ] No placeholders remaining
- [ ] Client reviewed the profile
- [ ] Client approved the profile
